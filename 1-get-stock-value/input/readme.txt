Note! For URL and script to work, name of stock must fit with URL. Example of none working name: KOBR MTFB. Remove or change script for this

Correct way to populate 0-allstocks.csv:
1. create new excel doc from inside this folder by right click
2. open file
3. Copy the content on SwedishBulls and return to doc
4. Paste (Match destination formatting) in top left
5. Choose save as
6. Save as type: CSV (Comma delimited); Choose Tools -> web options -> encoding, choose utf-8. Save. 
7. Upon closing prompt to save again, choose NO
8. Open document in Atom. There are chars here which need removing (<?>)
9. Search it and replace with nothing. Save
